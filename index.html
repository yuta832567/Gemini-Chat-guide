02
IMAGE GENERATION

画像生成：1日に作れる枚数

Geminiの画像生成は Nano Banana 2（標準・高速）と Nano Banana Pro（高品質）の2モデル構成。それぞれに別々の上限があります。

Nano Banana 2 と Nano Banana Pro
どちらもGoogleの画像生成エンジン。Nano Banana 2は日常使いの高速モデル、Nano Banana Proは高品質・高解像度モデルです。Nano Banana Proは正式名称「Gemini 3 Pro Image」として2026年6月に正式版（一般提供）になり、文字の描画精度に強みがあります。
例：SNS投稿用の画像 → Nano Banana 2／文字入りのポスターやクライアント提出資料 → Nano Banana Pro

リセットのタイミング

画像生成の枠は定期的にリセットされます。基準は太平洋時間（PT）で、日本時間では夕方16〜17時頃に復活するイメージです。Nano Banana Proの枠を超えた場合は、Nano Banana 2 に切り替えて生成を続けられます。

生成禁止カテゴリー（2026年2月強化）

実在の人物（公人・有名人）、版権キャラクター、顔交換（Face swap）、実在人物の衣装交換、金融文書の改ざん、これらに該当する画像はポリシー上生成がブロックされます。

CONTEXT WINDOW

コンテキストウィンドウ：一度に読める分量

「一度にどれだけの情報を理解できるか」を決める容量のことです。プラン差が大きいポイント。

HALLUCINATION

ハルシネーションはどれくらい起きるか？

UPDATED2026年のGemini 3系は事実性が大きく改善し、Gemini 3 Proは事実性ベンチマーク（SimpleQA Verified）で公開当時のトップ級スコアを記録しました。5月のGoogle I/O 2026で 3.5世代 も登場しています。ただしどのモデルでも誤り（ハルシネーション）は完全にはゼロになりません。

ハルシネーション
AIがもっともらしい嘘をつく現象。存在しない事実や誤情報を自信満々に生成してしまうこと。LLMが確率的に「次に来そうな言葉」を予測する仕組みに起因する構造的な問題です。
例：実在しない法律の条文を引用する、架空の人物の業績を作り出す、古い情報を最新として答える

Ultraプランの注意点

Ultraは強力ですが、データ保護の扱いはPro・無料版と同じです。機密情報を扱うなら、UltraではなくWorkspaceプラン（法人契約）を検討してください。補償プログラムもWorkspace/Vertex AIのみが対象です。
